# A3Intel Website

A modern, responsive website for A3Intel built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - React components for the website
  - `Header.tsx` - Navigation header with dropdown menus
  - `Announcements.tsx` - Announcement banner section
  - `Hero.tsx` - Hero section with main heading
  - `Services.tsx` - Services grid section
  - `Industries.tsx` - Industries showcase section
  - `Clients.tsx` - Clients section
  - `AwesomeFacts.tsx` - Statistics/facts section
  - `Contact.tsx` - Contact form and information
  - `Footer.tsx` - Footer with links and contact info

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ TypeScript for type safety
- ✅ Tailwind CSS for styling
- ✅ Modern Next.js 14 App Router
- ✅ Component-based architecture

## Build

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```
